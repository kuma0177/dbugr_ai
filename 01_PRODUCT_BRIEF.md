# Product Brief: debugr.ai

## One-liner

Turn native macOS screenshots and annotations into agent-ready product improvement tasks for Claude Code and Codex.

## Core Flow

```text
Open DMG app
→ choose browser page or another app experience
→ freeze screenshot with macOS picker
→ annotate boxes on the captured frame
→ confirm linked repo + Claude/Codex context
→ submit to agent
→ receive handoff feedback + next steps
→ review saved session in dashboard
```

## Target Users

- Product managers
- AI app builders
- Developers
- Designers
- QA teams
- Community managers
- Founders shipping fast
- Public beta communities

## Key Use Cases

### Native Product Feedback

A user opens the Mac app, freezes the exact browser tab or app view they want help with, draws annotations, and sends a structured feedback package to Claude Code or Codex.

### Repo-Aware Agent Handoff

Before sending, the user confirms that the capture belongs to the active Claude/Codex work and linked GitHub repo. The handoff response returns task context and next steps.

## MVP Goal

Build a working local platform where a user can:

1. Open the native macOS app.
2. Capture a browser tab or another on-screen experience.
3. Freeze a screenshot and annotate key regions.
4. Confirm the screenshot matches the active Claude/Codex work and linked repo.
5. Submit the session to Claude Code or Codex.
6. Receive immediate handoff feedback.
7. Review the saved session in the web dashboard.
8. Expose the feedback context through an MCP server.

## Non-goals for MVP

- Fully autonomous code changes.
- Full community/social feedback workflows.
- Perfect automatic AI-session detection.
- Native mobile app.
- Production billing.
