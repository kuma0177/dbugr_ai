# Product Brief: FeedbackAgent

## One-liner

Turn screen recordings, voice notes, screenshots, cursor metadata, and community feedback into agent-ready product improvement tasks.

## Core Flow

```text
Screen recording + voice notes
→ transcription + screenshots + cursor coordinates
→ structured feedback notes
→ AI-generated summary/task brief
→ public/private feedback thread
→ human/community review
→ direct input to Codex/Claude/Jira/GitHub/Figma or both
→ AI improvement task
→ agent/code/product update
```

## Target Users

- Product managers
- AI app builders
- Developers
- Designers
- QA teams
- Community managers
- Founders shipping fast
- Public beta communities

## Key Use Cases

### Screen Feedback

A user records their screen, explains an issue by voice, and the system creates structured feedback with transcript, screenshots, cursor coordinates, and implementation notes.

### Community Feedback on AI Output

An AI-generated result is published with a feedback thread. Humans comment, vote, suggest edits, or flag issues. The system clusters feedback and creates an improvement task.

### Agent Handoff

A reviewed feedback item is routed to Codex, Claude, ChatGPT, Gemini, Jira, GitHub Issues, Linear, or Figma.

## MVP Goal

Build a working local platform where a user can:

1. Create a feedback session.
2. Attach or simulate recording metadata.
3. Generate a mock transcript and AI summary.
4. Publish private or public feedback.
5. Comment and vote.
6. Create an improvement task.
7. Approve the task.
8. Send it to a mock integration.
9. Expose the feedback context through an MCP server.

## Non-goals for MVP

- Fully autonomous code changes.
- Full social platform integrations.
- Real-time multiplayer comments.
- Perfect computer vision UI detection.
- Native mobile app.
- Production billing.
